# Chat App Using Flask Backend (Python)

## Features

- Google Login
- Logout
- See All Users
- One To One Chat
- Delete Messages
- Responsive

## Backend

- Python
- Flask
- Jsonify
- Flask Socketio
- Mongodb (Mongoose)
- JWT
- Requests

## Frontend

- SCSS
- Redux Toolkit
- Socket.io Client
- Material UI
- Firebase (Google User Token)
- React Icons
- Material Icons
- React Redux
- Axios
- Momentjs
- Google Fonts
- React Router Dom
